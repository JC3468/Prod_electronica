Posteriormente, los archivos que nos descargo mods se envían a la Roland Monofab, una fresadora CNC utilizada para fabricar la placa en cobre. Este proceso consiste en la eliminación del material sobrante para dejar únicamente las pistas del circuito.

  • Preparación del material (placa de cobre)

    • Debes pegar la placa fenólica a la cama de sacrificio con cinta doble cara y asegurar que este bien pegada, ya que sino puede fallar el corte.

  • Configuración de la máquina Monofab

    • Conecta la MonoFab a tu computadora y abre el programa, de ahí pon la herramienta que corresponde y debes de cortar en el siguiente orden: 1. Perforaciones 2. pistas 3. Bordes El origen de los ejes x/y y z se ajusta al inicio, despues deberás de ir ajustando el origen del eje z de acuerdo a cada herramienta y entre cada proceso debes aspirar, para checar como va y vigilar que no se despegue.

3. Proceso de Soldado

Una vez fabricada la PCB, se procedió al montaje y soldado de los componentes electrónicos. Este paso requiere precisión y el uso de cautín, estaño y, en algunos casos, flux para mejorar la unión. Lo más importante a tener en cuenta es lo siguiente:

  • Colocación de componentes en la PCB
  • Soldadura de resistencias, capacitores, ICs y conectores
  • Revisión visual y pruebas eléctricas

Materiales y herramientas:

  • Cautín (30 a 60 W, preferiblemente con punta fina si es para electrónica).
  • Base o soporte para el cautín.
  • Estaño (aleación de plomo y estaño o libre de plomo).
  • Pasta o grasa para soldar (flux).
  • Esponja húmeda o limpiador de punta.
  • Pinzas o sujetadores.
  • Multímetro (opcional, para verificar continuidad).

Precedimiento:

  • Coloca el cautín en su base y conéctalo.
  • Espera de 3 a 5 minutos a que alcance la temperatura adecuada.
  • Humedece la esponja y colócala junto al soporte para limpiar la punta durante el proceso.
  • Limpia los componentes que vas a soldar (por ejemplo, las patas de un resistor o el cable de cobre).
  • Si hay óxido o suciedad, límpialos con una lija fina o alcohol isopropílico.
  • Aplica una pequeña cantidad de flux o grasa para soldar en las uniones; esto mejora la adherencia del estaño.
  • Toca con la punta del cautín la superficie metálica o el cable durante 1 o 2 segundos.
  • Aplica un poco de estaño hasta cubrirlo con una fina capa brillante.
  • Retira el cautín y deja enfriar.
  • Sujeta los componentes en su posición (por ejemplo, el cable sobre la terminal).
  • Calienta la unión con el cautín durante 1 a 2 segundos, sin aplicar estaño todavía.
  • Aplica el estaño en la zona calentada (no directamente sobre la punta del cautín).
  • Deja que el estaño se funda y cubra completamente la unión.
  • Retira el cautín y deja enfriar sin mover las piezas durante unos segundos. (Si queda opaca o con burbujas, puede haber una “soldadura fría”, que no conduce bien.)
  • Limpia la punta del cautín en la esponja húmeda para retirar residuos.
  • Corta los sobrantes de alambre o terminal con pinzas.
  • Si usaste flux en exceso, limpia los restos con alcohol isopropílico y un cepillo.

Verificación:

  • Revisa visualmente la soldadura (debe ser uniforme, sin grietas).
  • Si es un circuito, utiliza un multímetro para comprobar continuidad eléctrica.
  • Si hay falsos contactos o uniones débiles, repite el proceso calentando y aplicando un poco más de estaño.

Consejos:

  • No uses demasiado estaño; una pequeña cantidad es suficiente.
  • Mantén la punta del cautín siempre limpia y brillante.
  • Practica primero con cables o componentes viejos para mejorar tu técnica.

4. Multisim

Multisim es un entorno de simulación de circuitos que permite diseñar, probar y validar circuitos electrónicos antes de su montaje físico. A continuación se presenta un instructivo detallado, centrado en el uso de la interfaz, la colocación de componentes, las funciones clave y las configuraciones de simulación más útiles para prácticas de laboratorio.

Requisitos previos

  • Instalación de NI Multisim (NI Multisim 14.1).

Guía paso a paso (Instructivo)

  1. Interfaz y elementos principales
    • Barra de menús: File, Edit, Place, Simulate, Tools — aquí encontrarás todas las acciones principales.
    • Paleta de componentes: lista y búsqueda de componentes por nombre o categoría (Basic, Analog, Power, Digital).
    • Hoja de esquemático (workspace): área donde colocas y conectas componentes.
    • Barra de instrumentos: acceso rápido a osciloscopio, multímetro, generador de funciones y otros instrumentos virtuales.
    • Panel de propiedades: al seleccionar un componente muestra sus parámetros (valor, tolerancia, modelo SPICE).
  2. Paso 1 — Crear y guardar el proyecto
    • File → New → Design o Ctrl+N. Guardar: File → Save As; elige un nombre descriptivo (ej.: practica_amplificador.ms14).
    • Guardar frecuentemente para evitar pérdida de trabajo (autosave puede estar disponible según la versión).
  3. Paso 2 — Buscar e insertar componentes
    • Place → Component. Usa la búsqueda por texto si conoces el nombre (ej.: "resistor", "opamp" o "LM741").
    • Selecciona la librería adecuada: por ejemplo, Basic para resistencias y capacitores, Mixed para modelos SPICE de ICs.
    • Coloca el componente en la hoja haciendo clic en la posición deseada. Repite para añadir todos los componentes necesarios.
    • Si necesitas un símbolo que no aparece, revisa librerías adicionales o importa modelos SPICE específicos.
  4. Paso 3 — Editar parámetros de componentes
    • Haz doble clic en un componente para editar valor, tolerancia y propiedades avanzadas (modelos, pines, etc.).
    • Para resistencias y capacitores escribe valores legibles (10k, 1uF). Para fuentes, especifica amplitud, offset y forma de onda.
    • Algunos componentes permiten seleccionar modelos SPICE (útil para resultados más realistas, p. ej. modelos de transistores).
  5. Paso 4 — Cableado y organización del esquemático
    • Place → Wire o Ctrl+W. Une pines de componentes arrastrando desde un pin hasta otro.
    • Usa net labels (Place → Label) para nombrar nodos importantes: Vin, Vout, GND. Esto facilita lecturas y exportación de datos.
    • Organiza el esquemático en bloques funcionales (fuente, etapa de entrada, etapa de salida) para mayor claridad.
    • Si hay conexiones largas, usa puentes (Place → Terminal) o net labels para evitar wires cruzados.
  6. Paso 5 — Colocar instrumentos virtuales
    • Place → Instrument: selecciona Oscilloscope, Multimeter, Function Generator, Power Supply, Logic Analyzer según la prueba.
    • Conecta las puntas/probes del instrumento a los nodos que vas a medir (ej.: canal 1 del osciloscopio a Vout, referencia a GND).
    • Accede a las propiedades del instrumento (doble clic) para ajustar parámetros como escala, acoplamiento (DC/AC), y cursores.
  7. Paso 6 — Configurar tipos de simulación
    • Simulate → Interactive Simulation Settings. Elige el tipo de análisis según lo que busques:
    • Transient (Tiempo): para formas de onda en el dominio temporal. Parámetros recomendados: Stop time (ej.: 10 ms), Maximum time step (usa una fracción del periodo de la señal).
    • AC Analysis (Bode): para respuesta en frecuencia. Configura rango (ej.: 10 Hz – 1 MHz) y puntos por década.
    • DC Operating Point: para voltajes y corrientes en estado estacionario sin señales AC/temporal.
    • Parametric Sweep: (si la versión lo soporta) permite barrer valores (por ejemplo, variando R) y generar curvas múltiples automáticamente.
  8. Paso 7 — Ejecutar la simulación y usar herramientas de medida
    • Presiona F5 o el icono Run (▶) para iniciar la simulación.
    • En el osciloscopio: ajusta la escala vertical/horizontal, usa cursores para medir amplitud y periodo, y habilita múltiples canales para comparación.
    • En análisis AC: muestra magnitud en dB y fase; genera gráficas tipo Bode para identificar frecuencia de corte, ganancia y márgenes.
    • Utiliza el multímetro virtual para lecturas puntuales de DC (voltaje, corriente) o el analizador lógico para señales digitales.
  9. Paso 8 — Exportar datos e imágenes
    • Exportar traza: en la ventana del osciloscopio o del gráfico selecciona Trace → Export → CSV. Guarda con un nombre descriptivo (ej.: filtroRC_bode.csv).
    • Exportar imagen: File → Export → Bitmap o usa la herramienta de captura del sistema y guarda en imagenes/.
    • Incluye unidades y leyendas en las figuras para que sean legibles en el informe.
  10. Paso 9 — Funciones avanzadas y utilidades útiles
    • Parametric Sweep: automatiza variaciones de un componente y permite comparar resultados sin modificar el esquemático manualmente.
    • Monte Carlo: (si está disponible) analiza variaciones por tolerancias de componentes; útil para ver robustez del diseño.
    • SPICE Directives: permite incluir comandos SPICE personalizados para análisis específicos.
    • Model Import: importar modelos SPICE de fabricantes para obtener comportamiento realista de ICs y transistores.
    • Interactive Tuning: cambiar valores de componentes en tiempo real durante la simulación para ver efectos instantáneos.

Consejos prácticos y resolución de problemas

  • Si la simulación muestra resultados inesperados, revisa primero la conexión a GND y la referencia de las fuentes.
  • Para señales con mucho ruido aparente, reduce el máximo time step o añade condensadores de desacoplo en las fuentes de alimentación.
  • Si el osciloscopio no muestra nada, comprueba que el canal está habilitado y correctamente conectado al nodo y a tierra.
  • Documenta siempre la versión de Multisim; las rutas y nombres de menús pueden cambiar entre versiones.

5. Corte con Vinil – Máquina Brother ScanNCut DX

El corte con vinil permite crear etiquetas, logotipos y figuras decorativas de forma precisa. En esta práctica se utilizó la máquina Brother ScanNCut DX, la cual cuenta con un lector óptico integrado y diseños precargados que facilitan el trabajo sin necesidad de software externo. A continuación se explica el procedimiento paso a paso para realizar un corte usando un diseño predefinido.

Pasos a seguir

  1. Preparación del material y del área de trabajo
    • Verifica que la máquina esté conectada a la corriente y encendida correctamente.
    • Limpia la alfombrilla de corte (mat) para asegurar buena adherencia del vinil.
    • Corta un trozo de vinil del tamaño aproximado del diseño que deseas cortar.
    • Coloca el vinil sobre la alfombrilla con el lado brillante (superficie de color) hacia arriba y alísalo con la espátula para evitar burbujas.
  2. Encender y seleccionar diseño predefinido
    • Enciende la máquina con el botón Power ubicado en la parte lateral.
    • En la pantalla táctil, selecciona Patrón (Pattern).
    • Explora las categorías disponibles (formas básicas, letras, flores, bordes, etc.).
    • Elige un diseño predefinido, por ejemplo una figura geométrica o un logotipo sencillo.
    • Presiona OK para confirmar la selección.
  3. Ajustar el tamaño y la posición del diseño
    • Usa las opciones de la pantalla táctil para escalar el diseño (aumentar o reducir el tamaño).
    • Selecciona Editar → Mover para colocar el diseño dentro del área visible de la alfombrilla.
    • Verifica que el diseño no exceda los bordes del material adherido al mat.
  4. Colocar la alfombrilla en la máquina
    • Inserta la alfombrilla con el vinil en la ranura frontal de la máquina.
    • Presiona el botón Load Mat (icono con flecha hacia adentro) para que la máquina la reconozca.
    • En pantalla, aparecerá la vista previa del área de corte.
  5. Configurar el tipo de corte
    • Selecciona Corte (Cut) en el menú principal.
    • Asegúrate de que el tipo de material esté configurado como Vinil autoadhesivo.
    • Si tu máquina tiene sensor automático de cuchilla (Auto Blade), no es necesario ajustar la profundidad manualmente.
    • Si no, ajusta la cuchilla a una profundidad entre 1 y 2 según el grosor del vinil.
  6. Iniciar el proceso de corte
    • Verifica que todo esté correcto en la vista previa.
    • Presiona Start o el botón con el icono de triángulo para comenzar el corte.
    • La máquina cortará el diseño automáticamente, mostrando el progreso en la pantalla.
    • Evita tocar la alfombrilla o el material durante el proceso.
  7. Retirar y depilar el diseño
    • Cuando termine el corte, presiona Unload Mat para retirar la alfombrilla.
    • Usa una espátula o pinzas para levantar cuidadosamente el vinil sobrante, dejando solo el diseño deseado.
    • Si el diseño es muy detallado, retira con paciencia las partes pequeñas (interiores de letras o figuras).
  8. Aplicar el vinil
    • Coloca una hoja de transfer tape encima del diseño y presiona con la espátula para adherirlo.
    • Despega lentamente el vinil de la base y colócalo sobre la superficie final (por ejemplo, acrílico, metal o plástico).
    • Presiona con firmeza y retira la cinta de transferencia con cuidado.

Consejos prácticos

  • Siempre realiza una prueba de corte pequeña antes del diseño final para verificar la profundidad correcta de la cuchilla.
  • No uses vinil doblado o con residuos de polvo, ya que puede afectar la precisión del corte.
  • Después de varios usos, limpia la alfombrilla con agua tibia y jabón neutro para recuperar su adhesión.
  • Guarda los restos de vinil, pueden ser útiles para pruebas o cortes pequeños.

6. Uso y Manejo de la xTool

La xTool es una cortadora y grabadora láser de alta precisión ampliamente utilizada en prototipado rápido, fabricación de maquetas, corte de acrílico, grabado en maderas y diseño de componentes personalizados para proyectos de ingeniería. Su facilidad de uso y compatibilidad con múltiples formatos la convierten en una herramienta esencial dentro del laboratorio. A continuación se describe el proceso completo para utilizarla de forma segura, profesional y con resultados consistentes.

Pasos a seguir

  1. Preparación del material y del área de trabajo
    • Verifica que la máquina esté conectada a la corriente y al extractor de humo.
    • Revisa que la superficie de la base interna esté limpia de residuos carbonizados.
    • Coloca el material (acrílico, MDF, cartón, etc.) sobre la rejilla metálica y fija con imanes.
    • Asegúrate de que la tapa cierre por completo antes de encender el láser.
    • Para acrílico, retira solo la capa protectora superior; dejar la inferior reduce marcas por quemado.
    • Sugerencia de imagen: “imagenes/xTool/base_limpia.jpg” mostrando la cama de corte limpia.
    • Material recomendado: acrílico 3 mm, MDF 3 mm, cartón ilustración.
    • Archivo sugerido: “archivos/xtool/soporte_prototipo.dxf”.
  2. Configuración en xTool Creative Space
    • Abre el software e importa el archivo (.DXF, .SVG, .PNG, .JPG o .PDF).
    • Acomoda el diseño dentro del área de trabajo virtual.
    • Selecciona el tipo de proceso: Corte, Grabado o Puntuado.
    • Activa el modo de vista previa con cámara para verificar la posición del material real.
    • Sugerencia de imagen: “imagenes/xTool/interfaz_creative_space.jpg”.
    • Archivo editable: “archivos/xtool/logo_laser.svg”.
  3. Ajuste de parámetros de corte y grabado
    • Configura la potencia (Power) según el material: MDF 3 mm ≈ 90–100%.
    • Regula la velocidad (Speed) para obtener bordes limpios.
    • Define el número de pasadas (Passes) cuando el material requiere perforación completa.
    • Activa el Auto-Focus para que el láser ajuste su distancia automáticamente.
    • Sugerencia: incluir imagen “imagenes/xTool/material_settings.png” mostrando los presets.
  4. Vista previa y envío del trabajo
    • Haz clic en Framing para ver el contorno exacto que ocupará el diseño.
    • Verifica que no existan interferencias con imanes o bordes del material.
    • Da clic en Start para iniciar el corte.
    • No abras la tapa mientras el láser está en operación.
  5. Retiro del material y limpieza
    • Espera unos segundos tras finalizar para permitir que el humo se disipe.
    • Retira cuidadosamente las piezas y separa bordes con una espátula si es necesario.
    • Limpia el área y apaga la máquina.

Consejos prácticos

  • Evita cortar PVC o viniles que produzcan cloro al quemarse.
  • Para grabado más profundo, usa baja velocidad y potencia alta.
  • Para cortes limpios en acrílico: potencia alta + velocidad baja.
  • Guarda presets personalizados para proyectos futuros.

7. Uso y Manejo de Altium Designer

Altium Designer integra todas las etapas del diseño electrónico, desde la creación del esquemático hasta la generación de archivos de manufactura. Su enfoque profesional permite desarrollar PCB altamente confiables, con simulación eléctrica, reglas avanzadas y modelos 3D. A continuación se presenta el flujo completo con detalle técnico.

Preparar y crear un nuevo proyecto

    • Abre Altium designer
    • Crear un proyecto nuevo: File » New » Project → selecciona PCB Project y ponle nombre. Esto crea la estructura de proyecto (archivo .PrjPcb).
    • Dentro del proyecto, añade documentos: clic derecho en el proyecto (Project panel) → Add New to Project → Schematic para añadir un documento esquemático (.SchDoc) y PCB para añadir un documento de PCB (.PcbDoc).

      • Consejo: ve al Project Panel (normalmente a la izquierda) para ver y abrir los archivos del proyecto.

    • Archivo sugerido: “archivos/altium/esquematico_principal.PrjPcb”.
    • Sugerencia de imagen: “imagenes/Altium/schematic_closeup.jpg”.
  2. Esquemáticos
    • Abre un archivo .SchDoc (doble clic en Project Panel).
    • Añadir componentes: Place » Part o panel Components.
    • Cablear: Place » Wire. Para nombrar nets: Place » Net Label.
    • Power ports y buses: Place » Power Port, Place » Bus.
    • Vincular esquemático con PCB: Design » Update PCB Document <project>.PcbDoc (síncroniza netlist y componentes con la PCB).
    Tip: compila el proyecto (Project » Compile) para atrapar errores lógicos antes de actualizar la PCB.
  3. Librerías (componentes, huellas y gestión)
    • Tipos principales:
      • Schematic Library (.SchLib) — símbolos de esquemático.
      • PCB Library (.PcbLib) — huellas (footprints).
      • También existen bibliotecas integradas, bases de datos y el Altium Content Vault.
    • Crear librería esquemática: File » New » Library » Schematic Library. Para PCB: File » New » Library » PCB Library.
    • Gestionar componentes: paneles Libraries y Components; en las propiedades del componente puedes enlazar la huella, modelo 3D y modelos de simulación.
  4. Ruteado (Routing)
    • Abre el documento .PcbDoc.
    • Herramienta principal: Interactive Routing — Route » Interactive Routing o mediante atajo (según perfil del usuario).
    • Modos avanzados: Push-and-Shove, Hug, Walk-around, y Interactive Length Tuning (útil para señales de alta velocidad).
    • Presiona Tab mientras ruteas para editar propiedades de la pista (anchura, capa, restricciones).
  5. Etiquetas
    • En el esquemático: Place » Net Label para nombrar nets y permitir conexiones lógicas entre hojas.
    • Usa Place » Port y Place » Sheet Entry para conexiones entre múltiples hojas.
    • En PCB: visualiza y filtra nets en los paneles PCB Panel y PCB Inspector.
  6. PCB- Visualizar y preparar layout
    • Abre .PcbDoc. Usa la rueda del ratón para zoom; botón derecho abre el menú contextual.
    • Panels importantes: PCB (visibilidad de capas), PCB Rules and Violations, PCB Inspector, Components.
    • Editar stack-up: Design » Layer Stack Manager para definir capas, grosores y dielectricos.
    Nota: configura el Layer Stack antes del ruteado final — muchas reglas y cálculos (impedancia) dependen del stackup.
  7. Archivos de fabricación
    • Generar Gerber / NC Drill / ODB++: File » Fabrication Outputs » Gerber Files, NC Drill Files o ODB++ Files.
    • Usa un Output Job (.OutJob) para agrupar y versionar todas las salidas (Gerber, NC Drill, pick-and-place, BOM, drawings).
    • Archivos típicos que pide el fabricante: Gerbers (o Gerber X2), NC Drill (Excellon), archivo de stackup, y drawing de ensamblado si aplica.
  8. Simulación
    • Activa la extensión de simulación si hace falta (Extensions & Updates » Mixed-Signal Simulation).
    • En el esquemático, cada componente necesita un modelo de simulación (SPICE, Verilog-A, etc.) asignado en sus propiedades.
    • Panel de simulación: Simulation Dashboard (Panels » Simulation). Configura análisis (DC, AC, Transient) y puntos de sondeo; ejecuta con Simulate » Run Simulation.
    Tip: valida la netlist y modelos antes de correr simulación para evitar errores de asociación de modelos.
  9. Verificación DRC y manejo de reglas
    • Accede a las reglas: Design » Rules → se abre el PCB Rules and Constraints Editor.
    • Tipos de reglas comunes: Clearance, Width, Plane, Via, Component Placement, Electrical, etc.
    • Ejecutar DRC:
      • Online DRC: comprobaciones en tiempo real (si está habilitado).
      • Batch DRC: Tools » Design Rule Check o usa el panel PCB Rules And Violations para revisar la lista de violaciones.
    • Resolver violaciones: navega desde el panel de violaciones a los objetos afectados y corrige moviendo objetos o editando reglas.
  10. Flujo de trabajo recomendado (resumen práctico)
    • Crear proyecto → añadir esquemáticos.
    • Colocar componentes y nets → asignar huellas y verificar librerías.
    • Compilar esquemático (Project » Compile).
    • Actualizar PCB (Design » Update PCB Document), colocar componentes en el layout.
    • Definir reglas y stackup → ruteado (Interactive Routing) con Online DRC activo.
    • Simular el esquemático según necesidad (DC/AC/Transient).
    • Generar outputs de fabricación y revisar Gerbers antes de enviar al fabricante.
    11. Atajos, panels y dónde encontrarlos
    • Project Panel: lista archivos del proyecto (View » Panels » Project).
    • Components / Libraries: View » Panels » Components / Libraries.
    • Simulation Dashboard: Panels » Simulation.
    • PCB Rules And Violations: Panels » PCB Rules And Violations.
    Recursos
    • Revisa la documentación oficial de Altium para detalles de parámetros (Gerber X2, ajustes de ruteado, configuración de impedancia, etc.).
    • Opciones siguientes que puedo preparar para ti (elige una):
      • Checklist imprimible (PDF) con los pasos esenciales.
      • Tutorial paso a paso con capturas (adaptado a tu versión).
      • Ejemplo práctico: proyecto simple (esquemático + PCB) con comandos exactos para ejecutar en tu Altium.

8. Proyecto Fabricado

Para el proyecto final se integraron las herramientas descritas anteriormente para fabricar un prototipo funcional. La combinación de diseño electrónico en Altium y manufactura en la xTool permitió obtener un dispositivo completamente personalizado, optimizado y estéticamente profesional. Nuestro proyecto consta de un control remoto tipo los de Xbox.

Proceso completo

  1. Diseño electrónico
    • Se elaboró el esquemático del sistema completo.
    • Se seleccionaron componentes en función de requisitos eléctricos y disponibilidad.
    • Se verificaron tensiones, señales y compatibilidad de pines.
  2. Diseño de PCB
    • Se ruteó la tarjeta cuidando líneas críticas y zonas de potencia.
    • Se generaron Gerbers y se verificó su integridad.
    • Se hizo vista previa 3D para revisar ensamblaje.
  3. Fabricación de la placa
    • Se cortaron las pistas en la MonoFab al igual que las perforaciones.
    • En base a la calibración de la MonoFab que se puede ver en este mismo apartado.
  4. Integración y pruebas
    • Se soldaron los componentes en la PCB.
    • Se ensambló la tarjeta dentro del chasis.
    • Se realizaron pruebas funcionales, de comunicación y alimentación.
